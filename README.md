# 做了什麼優化

## 清除的雜訊（不影響程式碼內容）：

Pandoc 轉換殘留符號 — \[ \* \" \> 等逃脫字元全部還原成正常符號
重複的 > 來源： / > 說明： 區塊 — 619 條來源連結全部移除，只保留場景標題與程式碼
場景標題縮短 — 去掉冗長的 --- 這個腳本的作用是... 說明部分
多餘空行 — 連續 3 行以上空行壓縮為 1 行


## 保留的內容：

✅ 所有中文註解保留
